## Prerequisites

Before starting, ensure you have:

- **Python 3.8+** with pip
- **NeMo Run installed** (see [Installation Guide](../get-started/install.md))
- **Basic ML knowledge** (PyTorch, training loops)
- **Computing resources** (local machine or cloud access)
